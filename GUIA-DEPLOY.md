# 🚀 Guia Rápido de Deploy

## 📦 Passo 1: Preparar arquivos

### Baixe esta pasta para seu computador:
- **Opção A**: Clone do GitHub
- **Opção B**: Baixe o ZIP dos arquivos abaixo

### Arquivos principais:
```
src/
├── app/
│   ├── page.tsx      ← App principal
│   ├── layout.tsx
│   └── globals.css
├── components/ui/   ← Componentes
public/
├── manifest.json    ← Config do PWA
└── logo.svg          ← Ícone
package.json
next.config.ts
tailwind.config.ts
```

---

## ☁️ Passo 2: Criar repositório no GitHub

1. Acesse **github.com** e faça login
2. Clique em **"New repository"**
3. Nome sugerido: `treinamento-ia`
4. Deixe **público**
5. Clique em **"Create repository"**
6. Faça upload dos arquivos

---

## 🌐 Passo 3: Deploy no Vercel (GRÁTIS!)

1. Acesse **vercel.com**
2. Clique em **"Sign up with GitHub"**
3. Autorize o Vercel
4. Clique em **"New Project"**
5. Selecione seu repositório `treinamento-ia`
6. **Framework Preset**: Next.js (detecta automaticamente)
7. Clique em **"Deploy"**
8. Aguarde 1-2 minutos

**🎉 Pronto! Seu app estará online!**

---

## 📱 Passo 4: Instalar como App (SEM ADMIN!)

### Windows:
1. Abra o site no Chrome ou Edge
2. Clique no ícone de instalar ⬇️ na barra de endereço
3. Clique em **"Instalar"**
4. Pronto! App na área de trabalho!

### Android:
1. Abra no Chrome
2. Toque nos 3 pontos → "Adicionar à tela inicial"

### iPhone:
1. Abra no Safari
2. Toque em Compartilhar → "Adicionar a Tela de Início"
