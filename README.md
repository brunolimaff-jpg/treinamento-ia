# 🎓 Treinamento IA - PWA App

> **Aprenda Inteligência Artificial do zero até criar seus próprios agentes de IA**

## 🚀 Deploy no Vercel (GRATUITO)

### Passo 1: Fazer Upload no GitHub

1. Crie um repositório no GitHub
2. Faça upload de todos os arquivos desta pasta
3. Não esqueça de incluir a pasta `public/` e o arquivo `manifest.json`

### Passo 2: Conectar ao Vercel

1. Acesse [vercel.com](https://vercel.com)
2. Clique em "Sign up with GitHub"
3. Clique em "New Project"
4. Selecione seu repositório
5. Clique em "Deploy"

**Pronto!** 🎉

---

## 📱 Instalar como App (SEM ADMIN!)

### Windows (Chrome/Edge):
1. Abra o site no navegador
2. Clique no ícone ⬇️ na barra de endereço
3. Selecione "Instalar Treinamento IA"

### Android:
1. Chrome → 3 pontos → "Adicionar à tela inicial"

### iPhone:
1. Safari → Compartilhar → "Adicionar a Tela de Início"
